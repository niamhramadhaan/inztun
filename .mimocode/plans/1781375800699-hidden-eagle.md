# TypeScript Migration Plan

## Overview
Migrate the entire vanilla JS codebase (37 files) to strict TypeScript. No framework changes — just types.

## Step 1: Setup TypeScript Tooling
- Install `typescript` and `@types/node` as devDependencies
- Create `tsconfig.json` with `strict: true`, `"module": "ESNext"`, `"target": "ES2020"`, `"moduleResolution": "bundler"`
- Update `vite.config.js` → `vite.config.ts`
- Update `package.json` scripts to reference `.ts` entry

## Step 2: Create Shared Types
Create `src/types/index.ts` with shared interfaces:
- `Tool` — interface for tool classes (`name`, `icon`, `badge`, `render()`, `init(root)`, `destroy()`)
- `Route` — `{ module: string | null; tool: string | null }`
- `ToolRegistry` — `{ id, Tool, span, featured?, category?, categoryName? }`
- `Accent` — `{ hex: string; rgb: string }`
- `EventCallback` — `(data: any) => void`

## Step 3: Migrate Core Files (4 files)
- `src/core/events.js` → `src/core/events.ts` — type the EventBus Map, add generics for `on`/`emit`
- `src/core/router.js` → `src/core/router.ts` — type Router class, Route interface
- `src/core/db.js` → `src/core/db.ts` — type IndexedDB operations, generic store methods
- `src/core/cosmos.js` → `src/core/cosmos.ts` — type Canvas, star/nebula interfaces

## Step 4: Migrate Components (6 files)
- `src/components/Toast.js` → `src/components/Toast.ts`
- `src/components/Tile.js` → `src/components/Tile.ts` — type `TileOptions` interface
- `src/components/ToolView.js` → `src/components/ToolView.ts` — type options param
- `src/components/CommandPalette.js` → `src/components/CommandPalette.ts` — type Command interface
- `src/components/FloatingOrb.js` → `src/components/FloatingOrb.ts`
- `src/components/SettingsPanel.js` → `src/components/SettingsPanel.ts`

## Step 5: Migrate Workers Suite Module (22 files)
- `src/modules/workers-suite/index.js` → `src/modules/workers-suite/index.ts`
- `src/modules/workers-suite/tool-data.js` → `src/modules/workers-suite/tool-data.ts`
- All 20 tool files in `src/modules/workers-suite/tools/*.js` → `.ts`

## Step 6: Migrate Playground Module (8 files)
- `src/modules/playground/index.js` → `src/modules/playground/index.ts`
- `src/modules/playground/tool-data.js` → `src/modules/playground/tool-data.ts`
- All 6 tool files in `src/modules/playground/tools/*.js` → `.ts`

## Step 7: Migrate Entry Point
- `src/main.js` → `src/main.ts`
- Update `index.html` script tag to reference `src/main.ts`

## Step 8: Update All Imports
- Remove `.js` extensions from all import paths (TypeScript resolves without them)
- Ensure all cross-file imports use the correct `.ts` paths

## Step 9: Verify
- Run `npx tsc --noEmit` to confirm zero type errors
- Run `npm run build` to confirm Vite builds successfully
- Run `npm run dev` and verify the app loads correctly
